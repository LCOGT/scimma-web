This is SCiMMA-web. It demonstrates the use of the SCiMMA Hopskotch client library (source).
Here, you can add topics and event messages:
  • Create a topic.
  • Publish an event message to the SCiMMa server.
You can also list topics and event messages:
  • List the topics on SCiMMa server. From the list of topics, you can show its event messages.
  • List all the event messages.
  • See details of a given event message from the link on its topic page.
